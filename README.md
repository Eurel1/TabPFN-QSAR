# TabPFN-based QSAR Framework

## Overview
Environmental QSAR modeling is often constrained by small sample sizes, missing values, and class imbalance.  
This repository provides a unified framework based on the TabPFN family, integrating:

- Few-shot prediction (TabPFN)
- Missing-value robust inference (AutoTabPFN)
- Data utility curve–guided sampling strategy
- Generative augmentation (TabPFGen)
