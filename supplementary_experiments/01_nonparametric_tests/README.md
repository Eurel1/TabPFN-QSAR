# 01 - Non-parametric Statistical Tests

**Reviewer Concern:** "Reliance on visual comparisons of error bars [...] without formal non-parametric statistical hypothesis testing undermines the comparative claims of superiority." (Reviewer 4)

## What Was Done

We implemented a rigorous two-stage non-parametric testing protocol on 31 independent random seeds (40-70):

1. **Friedman test** - global comparison across all four models (TabPFN, XGBoost, CatBoost, SVM)
2. **Wilcoxon signed-rank test** - pairwise post-hoc comparisons with Bonferroni correction

## Files

| File | Description |
|------|-------------|
| nonparametric_tests.py | Main analysis script: Friedman + Wilcoxon with Bonferroni correction |

## Usage

```bash
python nonparametric_tests.py
```

Requires `model_comparison_results.csv` (generated by `model_compare.py` from the descriptor comparison experiment) in the same directory.
